# Adesi's Alyx ToolBox
This App Allows to Edit Soundevents files in your Addons using a GUI

## Features
- Loading Types directly from Pak
- Loading BaseSoundevents directly from Pak
- Detecting Alyx Directory
- Saving user Prefs
